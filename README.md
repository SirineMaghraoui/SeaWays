# SeaWays
A low poly 3D game in which the player controls and direct a ship sailing in the open seas trying to avoid obstacles by simulating/creating waves when interacting with the ocean using mouse clicks and drags. In this game the player interacts with the water surface mesh and changes the coordinates of the vertices.

# Tools I used:
- Unity D3
- C#
- Assets from 3D artists
